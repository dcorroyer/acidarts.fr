### Project acidarts.fr

Custom portfolio for my client.  
My client wanted to be able to create his projects himself.

##### Back-end:

    Symfony 4.4  
    PHP 7.2
    MySQL
    
##### Front-end:

    Bootstrap 4
    Twig
    
#### Structure of the application:

- Projects(home)
- Description(CV)
- Contact(mail)  
- Administration(hidden):
    - Login(security)
    - Project list
    - Project creation
    - Project modification
